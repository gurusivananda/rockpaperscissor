# Rock Paper Scissors Game 🎮

A simple web-based Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## 🌟 Features
- User vs Computer gameplay
- Score tracking
- Responsive design
- Fun visuals

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
